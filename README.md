# هنرآزمون (HonarAzmoon)

سامانه متن‌باز آزمون‌سازی و آزمون‌گیری مهارتی ویژه هنرستان‌ها

---

## معرفی پروژه

هنرآزمون با هدف تسهیل برگزاری آزمون‌های مهارتی، کاهش مصرف کاغذ، افزایش دقت در تصحیح آزمون‌ها و بهره‌گیری از فناوری در محیط‌های آموزشی طراحی می‌شود.

این سامانه به صورت آفلاین قابل اجرا بوده و برای استفاده در هنرستان‌ها، مدارس و مراکز آموزشی مناسب است.

---

## اهداف پروژه

- کاهش مصرف کاغذ در آزمون‌های تستی
- تصحیح خودکار آزمون‌ها
- امکان برگزاری آزمون بدون نیاز به اینترنت
- چاپ دفترچه آزمون
- چاپ پاسخنامه تشریحی نتایج
- توسعه یک سامانه متن‌باز آموزشی
- فراهم کردن بستر توسعه نسخه‌های آینده
